## 왜 다시 정리했는가

Java를 처음 공부할 때는 문법 하나하나에 집중하느라 입력과 출력의 흐름을 크게 보지 못할 때가 많았습니다.
이번에는 간단한 콘솔 프로그램을 만들면서 사용자 입력이 어떻게 변수에 저장되고, 그 결과가 다시 출력으로 연결되는지 한 번에 정리해보았습니다.

## 핵심 흐름

1. `Scanner` 객체를 생성한다.
2. 사용자의 입력을 문자열이나 정수 형태로 받는다.
3. 입력값을 조건문이나 연산에 활용한다.
4. 처리 결과를 `System.out.println()`으로 출력한다.

## 공부하며 느낀 점

처음에는 문법만 외우는 방식으로 접근했지만, 실제로는 "입력 -> 처리 -> 출력" 흐름으로 이해하는 편이 훨씬 오래 기억에 남았습니다.
앞으로도 기초 개념은 작은 예제와 함께 다시 정리하는 방식으로 학습을 이어가려고 합니다.
